## 1. 产品概述

HLK-LD2460 是一款用于人体感知的 24GHz 毫米波雷达产品，兼具距离角度测试功能，由两颗 1发2收 毫米波雷达芯片、高性能微带天线、高性能 MCU、高性能蓝牙 SOC 及外围辅助电路组成。采用 FMCW 波形，可准确识别 3-5 个人体目标，估计目标所在位置。

### 1.1 功能特性

| 项目 | 指标 |
|------|------|
| 检测人数 | 静态至少 3 人，运动最多 5 人 |
| 动态探测距离 | ±50° 内 ≥6m；±50°~±60° ≥5.5m |
| 静态探测距离 | ±50° 内 ≥5m；±50°~±60° ≥4.5m |
| 距离分辨率 | 0.75m |
| 距离精度 | ≤0.3m |
| 水平探测角度 | 120° |
| 垂直探测角度 | 90° |
| 角度分辨率 | 40° |
| 角度精度 | ≤5° |
| OTA 升级 | 支持 |
| 运动触发时间 | 0.5s |
| 存在感知时间 | ≤30s |

### 1.2 电气特性

| 项目 | 指标 |
|------|------|
| 工作频段 | 24GHz - 24.25GHz |
| 工作带宽 | 250MHz |
| EIRP | 13dBm |
| 调制方式 | FMCW（调频连续波） |

| 工作温度 | -40℃~85℃ |
| IO 电平 | 3.3V |

| PIN | 名称 | IO 类型 | 功能 |
|-----|------|---------|------|
| 1 | 5V | PWR | 电源输入 DC5V |

| 3 | Tx1 | O | 串口1 输出（调试用） |
| 4 | Rx1 | I | 串口1 输入（调试用） |

## 3. 安装方式

### 3.1 侧挂安装

- 模块与墙面夹角建议 25°~40°
- 建议安装高度 2.2~2.7m
- 需要通过上位机配置实际安装高度和角度

### 3.2 顶部安装

- 模块水平于墙面
- 建议安装高度 2.5~3m

## 4. 坐标系

### 4.1 侧装坐标系

- X 轴范围：[-6, 6]，雷达右方为正方向
- Y 轴范围：[0, 6]，雷达正前方为正方向
- 角度范围：[-60°, 60°]，法线方向为 0°，X 负方向为负角度

### 4.2 顶装坐标系

- X 轴范围：[-4, 4]，雷达右方为正方向
- Y 轴范围：[-4, 4]，雷达下方为正方向
- 角度范围：[0°, 360°]，X 正方向为 0°

## 5. 串口通信协议

### 5.1 通信参数

| 参数 | 值 |
|------|-----|
| 波特率 | 115200（出厂默认，可配置） |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | NONE |

### 5.2 帧格式

| 方向 | 帧头 | 帧尾 |
|------|------|------|
| 雷达 → 上位机（上报） | `F4 F3 F2 F1` | `F8 F7 F6 F5` |
| 上位机 → 雷达（命令） | `FD FC FB FA` | `04 03 02 01` |
| 雷达 → 上位机（回执） | `FD FC FB FA` | `04 03 02 01` |

帧结构：`帧头(4B) + 功能码(1B) + 数据长度(2B, 小端) + 数据内容(NB) + 帧尾(4B)`

---

## 6. 上报协议（功能码 0x04）

雷达检测到目标后主动上报，开机默认自动上报。

```
F4 F3 F2 F1 | 04 | [长度L] [长度H] | [目标1_X_L] [目标1_X_H] [目标1_Y_L] [目标1_Y_H] | ... | F8 F7 F6 F5
```

- 数据长度 = 目标数 × 4 + 11
- X/Y 坐标：int16 小端序，精度 0.1（实际值 = 原始值 × 0.1m）
- 示例：1 个目标 (1.5m, 2.3m) → `F4 F3 F2 F1 04 0F 00 0F 00 17 00 F8 F7 F6 F5`

---

## 7. 命令协议全量

### 7.1 开启/关闭上报（0x06）

| 操作 | 发送数据 | 回执数据 |
|------|----------|----------|
| 关闭上报 | `FD FC FB FA 06 0C 00 00 04 03 02 01` | `FD FC FB FA 06 0C 00 10 04 03 02 01` |
| 开启上报 | `FD FC FB FA 06 0C 00 01 04 03 02 01` | `FD FC FB FA 06 0C 00 11 04 03 02 01` |

回执格式：高4位=操作结果(1=成功)，低4位=操作内容(0=关闭,1=开启)

### 7.2 设置安装高度和角度（0x07）

仅侧装时有效，掉电保存。

```
发送: FD FC FB FA 07 0F 00 [高度L] [高度H] [角度L] [角度H] 04 03 02 01
回执: FD FC FB FA 07 0C 00 [01=成功/00=失败] 04 03 02 01
```
- 高度 = 原始值 / 100（米）
- 角度 = 原始值 / 100（度）

### 7.3 查询安装高度和角度（0x08）

```
查询: FD FC FB FA 08 0C 00 01 04 03 02 01
回执: FD FC FB FA 08 0F 00 [高度L] [高度H] [角度L] [角度H] 04 03 02 01
```

### 7.4 设置安装模式（0x09）

```
发送: FD FC FB FA 09 0C 00 [01=侧装/02=顶装] 04 03 02 01
回执: FD FC FB FA 09 0C 00 [高4位=结果, 低4位=模式] 04 03 02 01
```

### 7.5 查询安装模式（0x0A）

```
查询: FD FC FB FA 0A 0C 00 01 04 03 02 01
回执: FD FC FB FA 0A 0C 00 [01=侧装/02=顶装] 04 03 02 01
```

### 7.6 查询固件版本（0x0B）

```
查询: FD FC FB FA 0B 0C 00 01 04 03 02 01
回执: FD FC FB FA 0B 10 00 [模式] [年] [月] [主版本] [次版本] 04 03 02 01
```
示例：25年2月 V1.2 顶装 → `FD FC FB FA 0B 10 00 02 19 02 01 02 04 03 02 01`

### 7.7 重启雷达（0x0D）

```
发送: FD FC FB FA 0D 0C 00 01 04 03 02 01
```

### 7.8 设置波特率（0x0E）

| 序号 | 波特率 |
|------|--------|
| 0 | 9600 |
| 1 | 19200 |
| 2 | 38400 |
| 3 | 57600 |
| 4 | **115200**（出厂默认） |
| 5 | 230400 |
| 6 | 256000 |
| 7 | 460800 |

```
发送: FD FC FB FA 0E 0C 00 [序号] 04 03 02 01
回执: FD FC FB FA 0E 0C 00 [01=成功/00=失败] 04 03 02 01
```
掉电保存，重启后生效。

### 7.9 恢复出厂设置（0x10）

```
发送: FD FC FB FA 10 0C 00 01 04 03 02 01
回执: FD FC FB FA 10 0C 00 [01=成功/00=失败] 04 03 02 01
```

恢复默认值：波特率 115200、侧装、高度 2.6m、角度 30°、侧装检测 6m ±60°、顶装检测 4m 0-360°。

### 7.10 设置检测范围（0x11）

```
发送: FD FC FB FA 11 10 00 [距离] [起始角L] [起始角H] [终止角L] [终止角H] 04 03 02 01
回执: FD FC FB FA 11 0C 00 [01=成功/00=失败] 04 03 02 01
```
- 距离 = 原始值 / 10（米），侧装最大 6m，顶装最大 4m
- 角度 = int16 原始值 / 10（度），侧装 ±60°，顶装 0-360°

### 7.11 查询检测范围（0x12）

```
查询: FD FC FB FA 12 0C 00 01 04 03 02 01
回执: FD FC FB FA 12 10 00 [距离] [起始角L] [起始角H] [终止角L] [终止角H] 04 03 02 01
```

### 7.12 设置灵敏度（0x13，预留）

| 值 | 说明 |
|----|------|
| 0x01 | 灵敏度高 |
| 0x02 | 灵敏度中 |
| 0x03 | 灵敏度低 |

```
发送: FD FC FB FA 13 0C 00 [灵敏度值] 04 03 02 01
回执: FD FC FB FA 13 0C 00 [01=成功/00=失败] 04 03 02 01
```

### 7.13 查询灵敏度（0x14）

```
查询: FD FC FB FA 14 0C 00 01 04 03 02 01
回执: FD FC FB FA 14 0C 00 [灵敏度值] 04 03 02 01
```

## 9. 使用注意事项

- 开机稳定时间约 1 秒
- 电源要求：4.2V~5.4V，无明显毛刺或纹波，峰值电流 ≥180mA
- 串口需与模块共地
- 雷达电磁波可穿透衣物、窗帘、薄木板、玻璃
- 电风扇、宠物、植被、窗帘等运动物体可能引起误报
